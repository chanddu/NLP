NLP Project.

This project is being done as a part of the curriculum.

Work to be done (Part - A)

1.) Create a file data.txt which contains at least 500 reviews and their labels. 2.) Write code to remove stop words and convert the whole text to lowercase. Also drop any characters other than alphabets 3.) Show the vocabulary as a text file named vocabulary.txt. 4.) Train a multinomial naive bayes model. Use laplace smoothing/add one smoothing. 5.) Evaluation: Do 10 fold cross validation.
